# Results

This directory holds pipeline outputs generated by `scripts/run_full_pipeline.py`
(or by running the individual `src/models/*.py` scripts). It is not pre-populated
in the repository — running the pipeline will create:

| File | Produced by | Contents |
|---|---|---|
| `model_comparison_groupkfold.csv` | `src/models/train.py` | GroupKFold(5) R², RMSE, MAE for all six models |
| `model_comparison_temporal.csv` | `src/models/train.py` | Temporal holdout R², RMSE, MAE for all six models |
| `baseline_comparison.csv` | `src/models/train.py` | Naive, country FE-only, country FE+features baselines |
| `bootstrap_ci_temporal.json` | `src/models/validate.py` | Bootstrap 95% CI on XGBoost temporal R² |
| `loro_validation.csv` | `src/models/validate.py` | Leave-one-region-out R² and bootstrap CIs by region |
| `structural_break_test.json` | `src/models/validate.py` | Pre-/post-2015 Welch t-test and GroupKFold R² by period |
| `shap_importance.csv` | `src/models/explain.py` | Mean absolute SHAP value per feature (full dataset) |
| `shap_stability.csv` | `src/models/explain.py` | Multi-seed SHAP stability check |
| `permutation_importance.csv` | `src/models/explain.py` | Permutation importance per feature |
| `pdp.csv`, `ice.csv` | `src/models/explain.py` | Partial dependence and individual conditional expectation curves |
| `feature_ablation.csv` | `src/models/explain.py` | Temporal-holdout R² impact of dropping each feature |
| `horizon_sensitivity.csv` | `src/models/horizon_sensitivity.py` | GroupKFold and temporal R² for 3/5/7/10-year horizons |

The final, validated versions of these outputs (matching the published manuscript) are also available as static CSVs in `tables/`, with descriptive filenames matching the manuscript table numbers.
